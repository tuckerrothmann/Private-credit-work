# Source Log (Initial Build)

## How to read this log
- This is the **initial public-source pack** used for the first watchlist build.
- Some sources are direct company / sponsor / law-firm pages.
- Some are media summaries or republished Reuters-style items.
- I used these to identify issuers and patterns, **not** to claim exact lender exposures unless the source explicitly did so.

## Core market / framing sources

### S1. Morningstar — Why AI Worries About Software Are Hitting Private Credit
