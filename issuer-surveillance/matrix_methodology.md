# Issuer / Lender / BDC Matrix Methodology

## Purpose
The matrix is a surveillance tool, not a claim of confirmed portfolio holdings unless explicitly sourced.

## Cell meanings
- **confirmed exposure** = public source directly ties the issuer to the lender / BDC
- **relevant / high likelihood** = public financing evidence strongly suggests ecosystem relevance, even if vehicle-level holdings are not yet confirmed
- **plausible relevance** = name is large / visible enough that it should be checked against the lender platform
- **no evidence yet** = no meaningful public support found yet; continue checking

## What the matrix is trying to capture
1. Which issuers matter most for hidden mark risk
2. Which lenders are most likely to teach us something through public marks or disclosures
3. Where we have real evidence versus just a screening hypothesis
